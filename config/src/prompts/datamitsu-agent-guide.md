# datamitsu - AI Agent Guide

datamitsu is a tool for managing binaries and configs.

## Commands

- `datamitsu check` - Run fix and lint operations
- `datamitsu exec <app>` - Execute a managed tool
  - List apps: `datamitsu exec`
  - Pass args to app: `datamitsu exec <app> -- [app-args]`
