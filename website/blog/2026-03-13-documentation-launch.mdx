## What's included

The new documentation site covers everything you need to get started with datamitsu and use it effectively:

- **Getting Started** - Installation, quick start guide, and core concepts
- **Guides** - In-depth coverage of configuration, binary management, runtime management, managed configs, and the tooling system
- **How-To** - Task-oriented guides for common workflows like adding new tools, configuring linters, using remote configs, and managing cache
- **Reference** - CLI commands, configuration API, template placeholders, and ignore rules
- **Examples** - Real-world patterns including multiple tool versions, UV isolation, and PNPM patterns

## Get started

Head over to the [Getting Started](/docs/intro) guide to begin using datamitsu.
